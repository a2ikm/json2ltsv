# Json2ltsv

Convert JSON to LTSV line by line.

## Installation

Add this line to your application's Gemfile:

    gem 'json2ltsv'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install json2ltsv

## Usage

```
$ json2ltsv JSON_FILE > LTSV_FILE
```

or

```
$ cat JSON_FILE | json2ltsv > LTSV_FILE
```
